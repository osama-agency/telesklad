# Переменные окружения для Telegram ботов

## Основные настройки бота

### `TELEGRAM_BOT_TOKEN`
- **Описание**: Токен основного Telegram бота
- **Обязательная**: Да
- **Пример**: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`
- **Использование**: Основной токен для работы бота

### `TELEGRAM_BOT_USERNAME`
- **Описание**: Username бота в Telegram (с @)
- **Обязательная**: Да
- **Пример**: `@strattera_bot`
- **Использование**: Для формирования ссылок на бота

### `TELEGRAM_WEBHOOK_SECRET`
- **Описание**: Секретный токен для проверки подлинности webhook
- **Обязательная**: Да (для production)
- **Пример**: `my-secret-webhook-token-123`
- **Использование**: Проверка подписи запросов от Telegram

## Настройки администрирования

### `TELEGRAM_ADMIN_CHAT_ID`
- **Описание**: ID чата администратора для получения уведомлений
- **Обязательная**: Да
- **По умолчанию**: `125861752`
- **Использование**: Отправка системных уведомлений админу

### `TELEGRAM_COURIER_ID`
- **Описание**: ID курьера для отправки трек-номеров
- **Обязательная**: Нет
- **По умолчанию**: `821810448`
- **Использование**: Идентификация сообщений от курьера

### `TELEGRAM_ADMIN_IDS`
- **Описание**: Список ID администраторов через запятую
- **Обязательная**: Нет
- **По умолчанию**: `125861752`
- **Пример**: `125861752,987654321`
- **Использование**: Проверка прав доступа

### `TELEGRAM_TEST_ID`
- **Описание**: ID тестового чата для отладочных сообщений
- **Обязательная**: Нет
- **Использование**: Отправка тестовых уведомлений

## Настройки интерфейса бота

### `TELEGRAM_PREVIEW_MSG`
- **Описание**: Текст превью сообщения
- **Обязательная**: Нет
- **Использование**: Первое сообщение для новых пользователей

### `TELEGRAM_FIRST_VIDEO_ID`
- **Описание**: ID видео для приветствия
- **Обязательная**: Нет
- **Использование**: Отправка приветственного видео

### `TELEGRAM_BOT_BTN_TITLE`
- **Описание**: Текст кнопки для открытия приложения
- **Обязательная**: Нет
- **По умолчанию**: `Открыть приложение`
- **Использование**: Кнопка в сообщениях бота

### `TELEGRAM_GROUP_BTN_TITLE`
- **Описание**: Текст кнопки для присоединения к группе
- **Обязательная**: Нет
- **По умолчанию**: `Присоединиться к группе`
- **Использование**: Кнопка приглашения в группу

### `TELEGRAM_GROUP_URL`
- **Описание**: URL группы в Telegram
- **Обязательная**: Нет
- **По умолчанию**: `https://t.me/joinchat/your_group`
- **Использование**: Ссылка на группу

### `TELEGRAM_SUPPORT_URL`
- **Описание**: URL поддержки в Telegram
- **Обязательная**: Нет
- **По умолчанию**: `https://t.me/your_support`
- **Использование**: Ссылка на поддержку

## Бот для закупок

### `TELEGRAM_PURCHASE_BOT_TOKEN`
- **Описание**: Токен отдельного бота для закупок
- **Обязательная**: Нет
- **Использование**: Отправка уведомлений о закупках в группу

### `TELEGRAM_GROUP_CHAT_ID`
- **Описание**: ID группы для отправки закупок
- **Обязательная**: Нет
- **По умолчанию**: `-4729817036`
- **Использование**: Отправка закупок поставщику

## Токены для разных окружений

### `TELEGRAM_BOT_TOKEN_DEV`
- **Описание**: Токен бота для разработки
- **Обязательная**: Нет
- **Использование**: Автоматический выбор токена в dev режиме

### `TELEGRAM_BOT_TOKEN_PROD`
- **Описание**: Токен бота для production
- **Обязательная**: Нет
- **Использование**: Автоматический выбор токена в prod режиме

## Примечания

1. **Безопасность**: Никогда не коммитьте реальные токены в репозиторий
2. **Токены по окружению**: Используйте `TELEGRAM_BOT_TOKEN_DEV` и `TELEGRAM_BOT_TOKEN_PROD` для автоматического выбора токена
3. **Chat ID**: Для получения chat_id можно использовать бота @userinfobot
4. **Webhook Secret**: Генерируйте случайную строку для `TELEGRAM_WEBHOOK_SECRET`

## Пример конфигурации для разработки

```env
# Основной бот
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_BOT_USERNAME=@strattera_test_bot
TELEGRAM_WEBHOOK_SECRET=dev-secret-123

# Админ настройки
TELEGRAM_ADMIN_CHAT_ID=125861752
TELEGRAM_COURIER_ID=821810448

# UI настройки
TELEGRAM_BOT_BTN_TITLE=Открыть тестовое приложение
TELEGRAM_GROUP_URL=https://t.me/joinchat/test_group
TELEGRAM_SUPPORT_URL=https://t.me/test_support

# Бот закупок (опционально)
TELEGRAM_PURCHASE_BOT_TOKEN=987654321:ZYXwvuTSRqpONmlkJIHgfEDCba
TELEGRAM_GROUP_CHAT_ID=-1001234567890
```
